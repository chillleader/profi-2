**Задание 2**

Решение представлено в виде приложения Spring Boot.
Приложение разделено на модули (пакеты):
* controller - REST API контроллеры
* service - сервисный слой
* model - модели данных

В примере используется хранение в оперативной памяти в структуре
HashMap<id, Note>, однако интерфейс компонента, отвечающего
за хранение данных, сегрегирован, что позволяет заменить хранение
в оперативной памяти на хранение в любой базе данных
без изменений остального кода.

Для запуска неоюходимо собрать проект при помощи Maven (mvn package).

Документация реализована в формате Swagger.
Swagger UI URL: localhost:8080/swagger-ui.html
